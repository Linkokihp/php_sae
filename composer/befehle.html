Die wichtigsten Befehle im Terminal/Kommandozeile

Wo bin ich gerade?
Mac:
pwd
= print working directory

Windows:
cd

Welche Unterverzeichnisse gibt es hier?
Mac:
ls
= list

Windows:
dir

Verzeichnis hinauf
cd ..
= change directory

Verzeichnis hinunter
cd Ordnername
Es reicht aus, nur die ersten paar Buchstaben des Ordnernamens zu notieren und darauf die TAB-Taste zu drücken. (> Automatische Vervollständigung des Namens)

Checken, ob und welche Version von PHP vorhanden ist:
php -v oder php –version
Warum: Composer ist eigentlich ein PHP-Script, deshalb braucht es das als Voraussetzung.
